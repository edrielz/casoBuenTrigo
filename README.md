# Manual de Usuario - Panadería El Buen Trigo

## Guía Rápida de Inicio

### 1. Inicio de Sesión
- Ejecute el programa
- Ingrese usuario y contraseña
- Seleccione su rol correspondiente

### 2. Para Panaderos
- Registre recetas con ingredientes y cantidades
- Produzca panes seleccionando recetas existentes
- Consulte el stock de panes disponibles

### 3. Para Encargados de Inventario
- Mantenga actualizado el inventario de ingredientes
- Revise alertas de stock bajo
- Realice consultas del inventario actual

### 4. Para Administradores
- Genere reportes completos del sistema
- Exporte información a archivos de texto
- Monitoree el estado general de la panadería

## Consejos de Uso

- Siempre verifique el stock antes de producir
- Mantenga los niveles mínimos de ingredientes actualizados
- Exporte reportes regularmente para respaldo
- Utilice las alertas para planificar compras
